Social Media API
Description
This project is a Social Media API built using FastAPI and SQLAlchemy. It provides endpoints for various social media functionalities and utilizes a PostgreSQL database for data storage. As of the current version, user authentication has not been implemented yet.

Features
Built with FastAPI for efficient and fast development
Utilizes SQLAlchemy as an ORM for interacting with a PostgreSQL database
Provides endpoints for social media functionalities
